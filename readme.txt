Wtyczka umożliwiająca wysyłanie opinii oraz ich przeglądanie przez administratorów za pomocą bloków Gutenberga.

== Description ==
Wtyczka umożliwia dodanie bloku gutenberga z formularzem w którym przesyłamy opinie oraz bloku z lista naszych opini w którym możemy zobaczyć listę opini z ich szczegółowym podglądem. 

== Installation ==
1. Wgraj folder wtyczki do katalogu /wp-content/plugins/.
2. Aktywuj wtyczkę.
3. Dodaj bloki do treści posta lub strony.
